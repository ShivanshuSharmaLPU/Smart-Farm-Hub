# Smart Crop Advisory System

A comprehensive AI-powered agricultural advisory system helping farmers with crop & fertilizer recommendations, weather updates, yield prediction, and disease detection.

## Tech Stack
- **Frontend**: React.js (Vite, TailwindCSS)
- **Backend**: Node.js, Express.js, MongoDB
- **AI/ML**: Python, Flask, Scikit-learn
- **External API**: OpenWeatherMap

## Features
1. **Farmer Registration**: Sign up with location and soil details.
2. **Dashboard**: View real-time weather for your location.
3. **Crop Advisory**: Enter soil N-P-K values to get crop recommendations.
4. **Fertilizer Advisory**: Get fertilizer suggestions for your soil.
5. **Yield Prediction**: Estimate crop yield based on area and crop type.
6. **Disease Detection**: Mock AI detection for crop diseases.
7. **Multilingual**: Toggle between English and Hindi.
8. **Real ML Model**: Crop recommendation uses a trained Random Forest model on `Crop_recommendation.csv`.

## Manual Setup Instructions

If you prefer manual setup:

### 1. Database
Ensure MongoDB is running locally (`mongod`).

### 2. ML Service
```bash
cd ml_service
pip install -r requirements.txt
python train_models.py
python app.py
```
*Port: 5001*

### 3. Backend
```bash
cd backend
npm install
# Update .env with WEATHER_API_KEY if needed
npm run dev
```
*Port: 5000*

### 4. Frontend
```bash
cd frontend
npm install
npm run dev
```
*Port: 5173*

Open http://localhost:5173 in your browser.
